This plugin allows you to navigate a set of links layed out in a matrix from keyboard:

The rows should have the same vertical offset. You can mark one element with .selected class, or the first one can be selected automatically.

Pressing ENTER key navigates to the link in the current window.

Basic Usage:

$('#navigation a').keynav();

This will enable keyboard navigation for the links in #navigation container.

Advanced Usage:

$('#navigation a').keynav({ autoSelectFirst: false });

Please see possible options you can pass below.

Options

Option Default Comment
updateResizeInterval 500 Update the navigation matrix every 500ms when window is resized.
autoSelectFirst true Automatically select the first element (unless another element has .selected class)

API

$('a.links').keynav('update')

Update the keynav matrix manually

$('a.links').keynav('destroy')

Destroy the keynav and revert items to original state.
